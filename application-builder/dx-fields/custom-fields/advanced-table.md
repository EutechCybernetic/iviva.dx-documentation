**Advanced Table Options** \
This field type gives you detailed control over how tabular input works.

* **Option:** Customise each column's input type, width, and alignment
* **Purpose:** Allows to collect structured data with flexible input formats and dynamic row management
* **Use Case:** Ideal for inspection logs, resource matrices, onboarding checklists, and more
* **Functionality**
  * Set input type per column (_e.g., Text, Number, Email)_
  * Adjust alignment and width for each column
  * Choose whether users can add or remove rows
