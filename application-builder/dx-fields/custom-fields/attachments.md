This field type enables users to upload related media or documents from different sources.

**Tab:** Attachments\
**Purpose:** Controls what types of files users can upload (e.g., images, videos, documents)\
**Use Case:** Ideal for fields like 'Supporting Documents', 'Site Photos', or 'Proof of Work'\
**Functionality:**

* Toggle file pickers (e.g., Camera, Image, Video, File)
* Use selectors to set allowed input types
* If no specific option is set, all pickers are enabled by default.
