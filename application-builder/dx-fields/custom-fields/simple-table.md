This field type gives you a simple way to enter data in table form.

**Tab:** Table Options\
**Purpose:** Set up how table-style data is captured within the form\
**Use Case:** Great for listing items, entering logs, or capturing simple records _(e.g., Item, Quantity, Remarks)_\
**Functionality**

* **Table Headings:** You can define your own column titles _(e.g., Item, Quantity, Remarks)._
* **Read-Only Columns:** Optionally set certain columns so they cannot be edited
* **Row Control Toggles**
  * _Allow to Add Row:_ Lets users insert additional rows
  * _Allow to Remove Row:_ Enables users to delete rows
* **Max Length per Cell:**  **2048 characters**
