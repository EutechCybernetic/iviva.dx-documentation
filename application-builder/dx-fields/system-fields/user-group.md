#### ðŸ”¸ What is a User Group?

In **iviva DX**, a **User Group** is a collection of users who share similar responsibilities or need the same level of access to the system.

* **Represents**: Roles or teams like Admins, Supervisors, or Technicians
* **Used for**: Managing access, filtering responsibilities, and applying permissions
* **Assigned Roles**: Each group is linked to a user role that defines what features or modules its members can access

User Groups make it easier for administrators to manage permissions and organise system behaviour based on team responsibilities.
