# Defaults

#### **Defaults Tab**

The **Defaults** tab allows you to set predefined values for a field that automatically populate when the form is first loaded. This helps reduce manual input, guide user responses, and pre-fill commonly used data. Default values improve consistency and speed up the form-filling process, especially in high-volume or repetitive entry scenarios.
