# Map

Use the **Map** tab to set up how the Geo Map field works in your form. Also, you can connect to different map services and control how users interact with the map—making it easier to collect or display location data right in the app.

<figure><img src="../../.gitbook/assets/image (128).png" alt=""><figcaption></figcaption></figure>

**Component 1 → Map Type**\
Use the Map Type dropdown to choose the map service or style—like OpenStreetMap Standard, Google Maps Satellite, or ESRI World Imagery.

Your selection determines the map’s visual appearance and data source.

**Component 2 → Enable Search for Options**\
Turn the **Enable Search for Options** toggle on to let users search through map-related selections—like predefined places or areas—using a search box.\
It is especially useful when the list is long, helping users quickly find what they need.

**Component 3 → Allow to Pick Current Location**\
Enable the **Allow to Pick Current Location** toggle to allow users to capture their current physical location using device's GPS—ideal for mobile workflows or on-site submissions.

**Component 4 → Enable Zoom Control Buttons**\
Turn on the **Enable Zoom Control Buttons** toggle to show zoom in and zoom out buttons on the map. It gives users manual control over the zoom level, making navigation easier.

**Component 5 → Enable Interactive Zoom**\
Turn on the **Enable Interactive Zoom** toggle to let users zoom in and out of the map using gestures (like pinch-to-zoom) or mouse scroll. It improves usability and makes map navigation more intuitive.
