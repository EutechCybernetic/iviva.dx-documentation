# DX Composer Dashboard

### Dashboard Overview

The **DX Composer Dashboard** is a powerful tool designed to help users create, customise, and manage their own applications with ease. Whether you are building a simple app or a complex solution, the dashboard provides a user-friendly interface to design, test, and deploy your application. It offers a range of features including drag-and-drop components, customisable templates, real-time preview, and seamless integration with external services.&#x20;

The Dashboard allows users to create fully functional apps without the need for extensive coding knowledge, streamlining the app development process and accelerating time-to-market.

As shown in the image (DX Composer Dashboard) above, this page lists the applications you have already created, along with their available options that you should be aware of.

**Section 1: Side Panel**

* Use the side panel to navigate freely through the system.

**Section 2: Create Your Own App**

* This is the initial step for creating your own application.
* Click **Create Your Own App** to be directed to the page shown below.

<figure><img src="../.gitbook/assets/image (22).png" alt=""><figcaption><p>DX App Create/ App creation wizard</p></figcaption></figure>

**Section 3: App Icon**

* This is the app icon, and you can use an image relevant to the application.

**Section 4: App Status**

* Displays the current status of your application (_e.g. Inactive, Active_).

**Section 5: App Description**

* Shows a brief introduction to the app once it has been added.

**Section 6: App Version**

* &#x20;Displays the published version of your application.

**Section 7: Number of Workflows**

* &#x20;Shows the total number of workflows associated with your application.
