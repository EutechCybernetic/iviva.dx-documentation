### üõ†Ô∏è How to Build Dashboards Using Widgets

1. **Choose Your Widgets**\
   Select from a wide range of UXP widgets‚Äîcharts, summaries, layouts, or content panels‚Äîbased on what you want to display.
2. **Configure Widget Properties**
   * Set the data source (IvivaDX, API, Service, or Lucy)
   * Map fields, apply filters, and define thresholds
   * Enable behaviors like drill-downs, redirection, and conditional coloring
3. **Confirm and Place the widgets into the dashboard**\
   Arrange widgets in rows, columns, or grid structures using the dashboard layout editor.
4. **Iterate and Enhance**\
   Modify and improve your dashboard over time as data needs evolve‚Äîwithout any coding or re-deployment.

### UXP Dashboard Layout Overview

<figure><img src="../.gitbook/assets/1 - 2025-06-12T152154.679.png" alt=""><figcaption></figcaption></figure>

#### Component 1: Add Widget

This button lets you **add new widgets** to the dashboard.\
Clicking it opens a panel where you can choose from different widget types like charts, summary cards, layout maps, and more.\
Widgets help you build and customize your dashboard without any coding.

***

#### Component 2: Theme Toggle

This icon allows you to **switch between light and dark themes**.\
It's useful for adjusting the visual style of your dashboard depending on your preference or working environment.

***

#### Component 3: Profile and Language

Clicking the profile icon opens options to:

* View or update your **user profile**
* Change the **language** used in the dashboard\
  It helps personalize your dashboard experience.

***

#### Component 4: Widget Area (Main Dashboard)

This is the main space where **all widgets are placed**.\
You can:

* Add and arrange widgets
* View real-time data
* Interact with dashboard content\
  This area is the heart of your dashboard layout.

***

#### Component 5: Edit Dashboard(Pencil Icon)

The pencil icon appears on the dashboard bottom left. Clicking it allows you to:

* **Resize** the widget
* **Edit settings** of widgets.
* Access advanced options depending on the widget type\
  This is how you customize each widget to fit your needs.
