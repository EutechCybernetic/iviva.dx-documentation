# Add Sidebar Link

The **Add Sidebar Link** popup allows users to create custom navigation shortcuts in the DX Sidebar for easier access to frequently used resources or external platforms. Users can define a link title, specify a URL (internal or external), choose whether the link opens in a new tab, and upload a custom icon for visual identification.

<figure><img src="../../.gitbook/assets/1 (71) (1).png" alt=""><figcaption></figcaption></figure>

**Component 1:** **Link Title** – Enter a title for the link that will be displayed in the DX Sidebar menu. This helps users identify the purpose of the link at a glance.

**Component 2:** **Link** – Specify the full URL for the menu item. This can be either an internal application route or an external website.

**Component 3:** **Open in New Tab** – Select this checkbox if you want the link to open in a new browser tab instead of the current one.

**Component 4:** **Icon** – Choose or upload an icon to visually represent the menu item in the sidebar, allowing for quick visual recognition.
