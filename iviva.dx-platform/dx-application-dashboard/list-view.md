# List View

The List View presents all available applications in a vertical, scrollable format. Each application is displayed in a simplified row layout, making it easy to scan through app names and quickly perform actions such as creating new transactions or navigating to the search page.

\
**Components of the DX Application Dashboard (List View)**

* **Component 1 :** The side panel allows you to navigate through the system with ease.
* **Component 2 :** This is the search bar that lets you filter and quickly find apps by typing the app name.
* **Component 3:** Clicking anywhere on the application card, except the '+' will redirect you to the search page.
*  **Component 4:** Create transactions by clicking on the '+' icon
*  **Component 5:** This is a great feature that allows you to change the dashboard view with a single click. It allows you to switch between the grid view and the list view for the DX Application Dashboard.
