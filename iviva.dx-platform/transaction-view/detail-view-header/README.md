# Detail View Header

The **Detail View** page displays the complete information for a specific transaction within a DX application. While the overall structure of the page remains consistent, the **header section** may vary depending on configuration and use case.&#x20;

This page outlines the different types of detail view headers that can be encountered, including:

* **Detail View Header** – Displays the detail view header with all available configured options visible, except the Transition Bar.
* **Detail View Header with Transition Action Bar** – Adds a horizontal bar that enables quick transitions between workflow stages.
