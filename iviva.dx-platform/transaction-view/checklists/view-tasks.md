# View Tasks

This page displays all the tasks associated with a selected checklist. Each task is shown along with its **completion status**, **related comments**, and any available attachments. This allows users to easily track progress and review details for each task within the checklist.

<figure><img src="../../../.gitbook/assets/1 (50).png" alt=""><figcaption><p>View tasks</p></figcaption></figure>

* **Component 1 :** View Tasks of a Checklist\
  Clicking **View Tasks** for a specific checklist opens a popup displaying all tasks associated with that checklist. The popup includes the list of tasks, each task's completion status, associated comments, and any attachments (if available)<br>
