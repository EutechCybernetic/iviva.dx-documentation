# Checklist Complete

This section provides a clear interface for completing a checklist. To begin, click **View Tasks** to open this popup. Once all required tasks are completed, you can finalise the entire checklist by clicking the **Complete** button. \
The section uses intuitive **icons** and **status indicators** to guide users through task completion and checklist finalisation.

<figure><img src="../../../.gitbook/assets/1 (52).png" alt=""><figcaption><p>Checklist Complete</p></figcaption></figure>

* **Component 1 :** \
  Clicking this button marks the checklist as **complete.** Once completed, the **Complete** button changes to **Not Complete**, allowing the user to revert the status if needed or add additional information. The current status of the checklist is displayed visually, showing either **Completed** or **Not Completed**, along with a **green tick icon** if completed, or a **plain tick icon** if not completed, for a specific checklist, as shownin the image below.
