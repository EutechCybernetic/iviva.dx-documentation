# Basic Search

### Search by Transaction ID

* **Search by Transaction ID** is used to quickly locate a specific transaction by entering it in the search bar.

<figure><img src="../../.gitbook/assets/image (24).png" alt=""><figcaption><p>Search by Transaction ID</p></figcaption></figure>

### Search by Stage

* Find transactions based on their current stage by typing the stage name into the search bar.

<figure><img src="../../.gitbook/assets/image (25).png" alt=""><figcaption><p>Search by Stage Name</p></figcaption></figure>

### Search by Field Value

* Search using field values by entering the desired value in the search bar. This option is available only in **Basic Search**, and only for fields that have been configured with the "Include Field in Like Search" option.
