The **News Feed** in the iviva.DX mobile app shows important updates, announcements, and system messages from your organisation.

Each post includes:

* A **Title** and **category** _(e.g., API Development, Rest API_)
* A **Description** explaining the update
* The **Author** and **timestamp**
* Options to interact, such as **Like** and **Comment**

How to engage:

* Tap the **Thumbs-up** icon to like a post
* Use the speech bubble icon to view or add **comments**
* Type directly into the comment field to join the conversation.
